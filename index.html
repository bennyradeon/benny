Bab 1: Algoritma ‚Äì Kerangka Berpikir Komputer

Algoritma adalah urutan langkah logis untuk menyelesaikan masalah. Ibarat resep masakan: tanpa resep, koki bisa salah. Tanpa algoritma, program bisa kacau.

Contoh algoritma menghitung luas persegi panjang:

1. Mulai
2. Masukkan panjang (P)
3. Masukkan lebar (L)
4. Hitung luas = P * L
5. Tampilkan luas
6. Selesai

Nanti kita akan terjemahkan ke Python. Fokus kita di modul ini adalah pemrograman Python, tapi selalu ingat algoritma di belakangnya.

Soal 1.1

Buat algoritma untuk:

  1. Mencari bilangan terbesar dari tiga angka.
  2. Menghitung rata-rata tiga nilai ujian.
  3. Menentukan apakah suatu tahun adalah tahun kabisat.

Jawaban:

1. Algoritma mencari bilangan terbesar dari tiga angka:
1. Mulai
2. Masukkan a, b, c
3. Jika a > b dan a > c maka terbesar = a
4. Jika b > a dan b > c maka terbesar = b
5. Jika c > a dan c > b maka terbesar = c
6. Tampilkan terbesar
7. Selesai

2. Algoritma rata-rata tiga nilai:
1. Mulai
2. Masukkan nilai1, nilai2, nilai3
3. jumlah = nilai1 + nilai2 + nilai3
4. rata = jumlah / 3
5. Tampilkan rata
6. Selesai

3. Algoritma tahun kabisat:
1. Mulai
2. Masukkan tahun
3. Jika tahun % 400 == 0 maka kabisat = benar
4. Jika tahun % 100 == 0 maka kabisat = salah
5. Jika tahun % 4 == 0 maka kabisat = benar
6. Jika tidak, kabisat = salah
7. Tampilkan "kabisat" atau "bukan"
8. Selesai

2.1 Variabel dan Tipe Data ‚Äì Kotak Penyimpanan

Variabel adalah nama yang mewakili suatu nilai di memori. Python bersifat dynamically typed: kita tidak perlu menyebutkan tipe data, Python akan menentukannya dari nilai yang diberikan.

Contoh 1: Membuat variabel

nama = "Dewi"           # string (teks)
usia = 17               # integer (bilangan bulat)
tinggi = 162.5          # float (bilangan desimal)
menikah = False         # boolean (True/False)

print(nama)
print(usia)
print(tinggi)
print(menikah)

Apa yang terjadi jika kita mengganti nilai? Variabel bisa diubah kapan saja, bahkan dengan tipe berbeda. Tapi hati-hati, bisa membingungkan.

usia = 17
usia = "tujuh belas"    # sekarang usia menjadi string (tidak disarankan)

Mengenal Tipe Data Lebih Dalam

TipeContohKegunaan
int25, -3, 0Bilangan bulat
float3.14, -0.5, 2.0Bilangan desimal
str"Halo", 'A'Teks (string)
boolTrue, FalseKebenaran logika

Kita bisa cek tipe data dengan fungsi type():

print(type(nama))   # 
print(type(usia))   #

Konversi Tipe Data (Casting)

Kadang kita perlu mengubah tipe, misal dari string ke integer agar bisa dihitung.

angka_str = "100"
angka_int = int(angka_str)   # mengubah string ke integer
print(angka_int + 50)         # 150

nilai_float = float("75.5")   # jadi float
print(nilai_float)
üí° Catatan: Jika string tidak bisa diubah ke angka (misal "seratus"), akan terjadi error. Selalu pastikan input sesuai.

2.2 Input dan Output ‚Äì Berkomunikasi dengan Pengguna

Output: menggunakan print() untuk menampilkan teks atau nilai ke layar.

Input: menggunakan input() untuk meminta pengguna mengetik sesuatu. Hasil input() selalu bertipe string.

Contoh 2: Program sapa sederhana

nama = input("Siapa nama kamu? ")            # user mengetik nama
print("Halo, " + nama + "! Selamat belajar.")   # penggabungan string

Penjelasan: input() akan menampilkan teks di dalam tanda kurung, lalu menunggu user mengetik dan menekan Enter. Apa yang diketik disimpan ke variabel nama.

Bagaimana jika ingin input angka? Harus dikonversi.

usia = int(input("Masukkan usia Anda: "))     # langsung di-casting
print("Tahun depan usia Anda", usia + 1)

Alternatif: dua baris

usia_str = input("Masukkan usia: ")
usia = int(usia_str)

Format Output yang Rapi

Kita bisa menggunakan f-string (format string) untuk menyisipkan variabel ke dalam teks dengan mudah.

nama = "Budi"
umur = 16
print(f"Nama saya {nama}, umur {umur} tahun.")  # lebih rapi

f-string diawali huruf f sebelum tanda petik, dan variabel ditulis dalam kurung kurawal {}.

Input Banyak Nilai Sekaligus

Kita bisa meminta beberapa input berturut-turut, atau menggunakan split() untuk memisahkan input dalam satu baris.

# Meminta dua angka dalam satu baris (dipisah spasi)
data = input("Masukkan dua angka (pisah spasi): ")
a, b = data.split()      # split() menghasilkan list, lalu kita unpack
a = int(a)
b = int(b)
print(f"Jumlah: {a+b}")

2.3.3 Operator Logika (and, or, not)

Digunakan untuk menggabungkan kondisi boolean.

  • and : True jika kedua operand True.
  • or : True jika salah satu True.
  • not : membalikkan nilai (True jadi False, sebaliknya).
a = True
b = False
print(a and b)   # False
print(a or b)    # True
print(not a)     # False

# Contoh dalam kondisi
nilai = 85
absen = 90
if nilai >= 80 and absen >= 80:
    print("Lulus dengan baik")

2.3.4 Operator Penugasan (Assignment)

Selain =, ada operator gabungan seperti +=, -=, dll.

c = 10
c += 5   # sama dengan c = c + 5 -> 15
c -= 3   # c = c - 3 -> 12
c *= 2   # 24
c /= 4   # 6.0 (float)
print(c)

2.3.5 Eksplorasi: Apa yang Terjadi Jika Operator Diganti?

Pertanyaan: Mengapa ada dua jenis pembagian (/ dan //)?

Jawab: / selalu menghasilkan float (desimal), sedangkan // menghasilkan pembulatan ke bawah (integer). Pilih sesuai kebutuhan: jika ingin hasil presisi, gunakan /; jika ingin hasil bulat (misal untuk indeks), gunakan //.

Pertanyaan: Apa bedanya = dan ==?

Jawab: = adalah penugasan (assignment), misal x = 5. == adalah perbandingan kesamaan. Salah satu kesalahan umum pemula adalah menulis if x = 5 (error) seharusnya if x == 5.

Bab 3: Percabangan ‚Äì Membuat Keputusan

Dengan if, elif, else, program dapat memilih tindakan berdasarkan kondisi.

3.1 If Tunggal

Contoh: Cek apakah suatu bilangan positif.

angka = int(input("Masukkan bilangan: "))
if angka > 0:
    print("Bilangan positif")

Jika angka <=0, tidak ada output.

3.2 If-Else ‚Äì Dua Cabang

angka = int(input("Masukkan bilangan: "))
if angka % 2 == 0:
    print("Genap")
else:
    print("Ganjil")

Mengapa pakai else? Karena bilangan pasti genap atau ganjil. Else mencakup semua kemungkinan yang tidak memenuhi if.

3.3 If-Elif-Else ‚Äì Banyak Kemungkinan

nilai = int(input("Masukkan nilai: "))
if nilai >= 90:
    grade = "A"
elif nilai >= 80:
    grade = "B"
elif nilai >= 70:
    grade = "C"
elif nilai >= 60:
    grade = "D"
else:
    grade = "E"
print(f"Grade: {grade}")

Perhatikan urutan: dari yang paling tinggi ke rendah. Jika dibalik, logikanya salah.

3.4 Nested If (If di dalam If)

Contoh: Menentukan jenis segitiga.

a = float(input("Sisi a: "))
b = float(input("Sisi b: "))
c = float(input("Sisi c: "))
if a + b > c and a + c > b and b + c > a:
    if a == b == c:
        print("Segitiga sama sisi")
    elif a == b or b == c or a == c:
        print("Segitiga sama kaki")
    else:
        print("Segitiga sembarang")
else:
    print("Bukan segitiga")

Pertama cek syarat segitiga, baru kemudian jenisnya.

Bab 4: Perulangan ‚Äì Melakukan Pekerjaan Berulang

4.1 While Loop

Selama kondisi benar, blok kode dijalankan terus.

# Menampilkan angka 1 sampai 5

    print(i)

for i in range(1, 6):    # 1,2,3,4,5
    print(i)

for i in range(2, 11, 2): # 2,4,6,8,10

4.3 Break dan Continue

break menghentikan loop, continue melompat ke iterasi berikutnya.

# Mencari angka pertama yang habis dibagi 7 antara 1-50
for i in range(1,51):
    if i % 7 == 0:
        print(f"Ketemu: {i}")
        break

# Melewati angka genap
for i in range(1,11):
    if i % 2 == 0:
        continue
    print(i)   # hanya ganjil

Bab 6: Proyek Akhir ‚Äì Aplikasi Pengelolaan Nilai Siswa

Program ini akan menyimpan data siswa (nama, nilai matematika, ipa, inggris) dalam list of dictionaries. Fitur: tambah, tampilkan semua, cari, dan nilai tertinggi.

data_siswa = []

def tambah():
    n = input("Nama: ")
    mtk = float(input("Matematika: "))
    ipa = float(input("IPA: "))
    ing = float(input("Inggris: "))
    data_siswa.append({"nama": n, "mtk": mtk, "ipa": ipa, "ing": ing})
    print("Data tersimpan.")

def tampil():
    if not data_siswa:
        print("Belum ada data.")
        return
    for s in data_siswa:
        rata = (s['mtk'] + s['ipa'] + s['ing']) / 3
        print(f"{s['nama']} | MTK:{s['mtk']} IPA:{s['ipa']} ING:{s['ing']} | Rata: {rata:.2f}")

def cari():
    nama = input("Nama yang dicari: ")
    for s in data_siswa:
        if s['nama'].lower() == nama.lower():
            rata = (s['mtk'] + s['ipa'] + s['ing']) / 3
            print(f"Ditemukan: {s['nama']} - Rata: {rata:.2f}")
            return
    print("Tidak ditemukan.")

def tertinggi():
    if not data_siswa:
        print("Belum ada data.")
        return
    terbaik = max(data_siswa, key=lambda s: (s['mtk']+s['ipa']+s['ing'])/3)
    rata = (terbaik['mtk']+terbaik['ipa']+terbaik['ing'])/3
    print(f"Nilai tertinggi: {terbaik['nama']} dengan rata {rata:.2f}")

def menu():
    while True:
        print("\n=== APLIKASI NILAI SISWA ===")
        print("1. Tambah data")
        print("2. Tampilkan semua")
        print("3. Cari siswa")
        print("4. Nilai tertinggi")
        print("5. Keluar")
        p = input("Pilih: ")
        if p == '1': tambah()
        elif p == '2': tampil()
        elif p == '3': cari()
        elif p == '4': tertinggi()
        elif p == '5': break
        else: print("Pilihan salah.")

if __name__ == "__main__":
    menu()

            data_siswa = json.load(f)
        print("Data dimuat dari file.")
    except FileNotFoundError:
        print("File tidak ditemukan.")

Bab 7: Array (List) ‚Äì Menyimpan Banyak Data dalam Satu Variabel

Array dalam Python disebut list. List adalah struktur data yang dapat menyimpan banyak nilai (elemen) dalam satu variabel. Elemen-elemen dalam list dapat diakses menggunakan indeks (nomor urut), dan list bersifat mutable (dapat diubah).

7.1 Membuat List

List dibuat dengan tanda kurung siku [] dan elemen dipisahkan dengan koma.

# List kosong
list_kosong = []

# List dengan beberapa elemen
buah = ["apel", "mangga", "jeruk", "durian"]
angka = [10, 20, 30, 40, 50]
campuran = ["teks", 123, True, 3.14]  # list bisa berisi campuran tipe data

print(buah)
print(angka)
print(campuran)

7.2 Mengakses Elemen List

Indeks dimulai dari 0 untuk elemen pertama. Indeks negatif dimulai dari -1 untuk elemen terakhir.

buah = ["apel", "mangga", "jeruk", "durian"]
print(buah[0])    # apel
print(buah[2])    # jeruk
print(buah[-1])   # durian (elemen terakhir)
print(buah[-2])   # jeruk (elemen kedua dari belakang)

7.3 Mengubah Nilai Elemen

Karena list bersifat mutable, kita bisa mengubah nilai elemen tertentu.

buah = ["apel", "mangga", "jeruk", "durian"]
buah[1] = "anggur"   # mengubah mangga menjadi anggur
print(buah)          # ['apel', 'anggur', 'jeruk', 'durian']

7.4 Operasi Dasar List

Menambahkan Elemen

  • append(x) : menambahkan elemen x di akhir list
  • insert(i, x) : menyisipkan elemen x pada posisi i
  • extend(iterable) : menambahkan semua elemen dari iterable (misal list lain) ke akhir list
buah = ["apel", "mangga"]
buah.append("jeruk")        # ['apel', 'mangga', 'jeruk']
buah.insert(1, "anggur")     # ['apel', 'anggur', 'mangga', 'jeruk']
buah.extend(["durian", "nanas"])  # ['apel', 'anggur', 'mangga', 'jeruk', 'durian', 'nanas']
print(buah)

Menghapus Elemen

  • remove(x) : menghapus elemen pertama yang bernilai x
  • pop(i) : menghapus elemen pada indeks i (jika i tidak diberikan, hapus elemen terakhir) dan mengembalikan nilai yang dihapus
  • clear() : menghapus semua elemen
  • del list[i] : menghapus elemen pada indeks i
buah = ["apel", "mangga", "jeruk", "durian", "mangga"]
buah.remove("mangga")        # menghapus mangga pertama -> ['apel', 'jeruk', 'durian', 'mangga']
buah.pop(2)                  # menghapus elemen indeks 2 (durian) -> ['apel', 'jeruk', 'mangga']
buah.pop()                   # menghapus elemen terakhir (mangga) -> ['apel', 'jeruk']
del buah[0]                  # menghapus elemen pertama (apel) -> ['jeruk']
buah.clear()                 # [] (kosong)

Mencari Elemen

  • index(x) : mengembalikan indeks pertama dari elemen x (error jika tidak ditemukan)
  • count(x) : mengembalikan jumlah kemunculan x
  • in : operator untuk mengecek apakah suatu nilai ada dalam list
buah = ["apel", "mangga", "jeruk", "mangga"]
print(buah.index("mangga"))    # 1
print(buah.count("mangga"))    # 2
print("apel" in buah)          # True
print("durian" in buah)        # False

Mengurutkan dan Membalik

  • sort() : mengurutkan list secara ascending
  • reverse() : membalik urutan list
  • sorted(list) : fungsi bawaan yang mengembalikan list baru yang sudah terurut
angka = [5, 2, 8, 1, 9]
angka.sort()                 # [1, 2, 5, 8, 9]
print(angka)

angka.reverse()              # [9, 8, 5, 2, 1]
print(angka)

# Tanpa mengubah list asli
angka2 = [3, 1, 4, 2]
print(sorted(angka2))        # [1, 2, 3, 4]
print(angka2)                # masih [3, 1, 4, 2] (tidak berubah)

Menyalin List

Perhatikan bahwa assignment biasa (list2 = list1) tidak membuat salinan baru, hanya referensi. Gunakan copy() atau list[:] untuk menyalin.

list1 = [1, 2, 3]
list2 = list1          # ini bukan salinan, hanya referensi
list2.append(4)
print(list1)           # [1, 2, 3, 4] (ikut berubah)

# Cara benar menyalin
list3 = list1.copy()
list4 = list1[:]
list3.append(5)
print(list1)           # [1, 2, 3, 4] (tidak berubah)
print(list3)           # [1, 2, 3, 4, 5]

7.5 List Comprehension

Cara ringkas untuk membuat list baru berdasarkan list yang sudah ada.

# Membuat list kuadrat dari 1-5
kuadrat = [x**2 for x in range(1, 6)]
print(kuadrat)        # [1, 4, 9, 16, 25]

# Hanya bilangan genap dari 1-10
genap = [x for x in range(1, 11) if x % 2 == 0]
print(genap)          # [2, 4, 6, 8, 10]

# Mengubah huruf menjadi kapital
buah = ["apel", "mangga", "jeruk"]
buah_kapital = [b.upper() for b in buah]
print(buah_kapital)   # ['APEL', 'MANGGA', 'JERUK']

7.6 Iterasi List dengan Loop

buah = ["apel", "mangga", "jeruk"]

# Cara 1: for langsung
for b in buah:
    print(b)

# Cara 2: menggunakan indeks
for i in range(len(buah)):
    print(f"Indeks {i}: {buah[i]}")

# Cara 3: dengan enumerate (dapat indeks dan nilai)
for i, b in enumerate(buah):
    print(f"{i}: {b}")

7.7 Studi Kasus: Mengelola Daftar Nilai Mahasiswa

Program sederhana untuk menyimpan nilai ujian, menghitung rata-rata, nilai tertinggi, terendah.

# Input nilai
nilai = []
while True:
    input_nilai = input("Masukkan nilai (atau ketik 'selesai'): ")
    if input_nilai.lower() == 'selesai':
        break
    try:
        nilai.append(float(input_nilai))
    except:
        print("Nilai tidak valid, masukkan angka.")

if nilai:
    print(f"Daftar nilai: {nilai}")
    print(f"Rata-rata: {sum(nilai)/len(nilai):.2f}")
    print(f"Nilai tertinggi: {max(nilai)}")
    print(f"Nilai terendah: {min(nilai)}")
else:
    print("Tidak ada nilai yang dimasukkan.")

Soal 7.6 ‚Äì Matriks Sederhana (List of Lists)

Buat list 2 dimensi (list di dalam list) yang merepresentasikan matriks 3x3. Minta user mengisi nilai untuk setiap elemen. Kemudian:

Soal 7.7 ‚Äì Gabungan dengan Fungsi

Buat fungsi berikut:

  • rata_rata(list_angka) : mengembalikan rata-rata
  • nilai_maksimal(list_angka) : mengembalikan nilai maks (tanpa max bawaan)
  • nilai_minimal(list_angka) : mengembalikan nilai min (tanpa min bawaan)

Kemudian dalam program utama, minta user memasukkan 5 angka, panggil ketiga fungsi tersebut dan tampilkan hasilnya.

Jawaban:

def rata_rata(lst):
    return sum(lst) / len(lst) if lst else 0

def nilai_maksimal(lst):
    if not lst:
        return None
    maks = lst[0]
    for n in lst:
        if n > maks:
            maks = n
    return maks

def nilai_minimal(lst):
    if not lst:
        return None
    min = lst[0]
    for n in lst:
        if n < min:
            min = n
    return min

# Program utama
angka = []
for i in range(5):
    n = float(input(f"Masukkan angka ke-{i+1}: "))
    angka.append(n)

print(f"Angka: {angka}")
print(f"Rata-rata: {rata_rata(angka):.2f}")
print(f"Nilai maksimal: {nilai_maksimal(angka)}")
print(f"Nilai minimal: {nilai_minimal(angka)}")
